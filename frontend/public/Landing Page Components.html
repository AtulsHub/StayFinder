Great idea, Atul! Here's how you can add a calendar component (DateCalendar) above the map section, that:

  1. Shows crossed-out dates that already have bookings.

  2. Allows the user to select a date, which:

    • Updates the selected date in the booking section.

    • If the user updates the date in the booking form, it updates the calendar selection too (i.e., 2-way sync).

Step-by-step plan:

1. Install react-calendar (if not already installed):

bash
npm install react-calendar

2. Create the DateCalendar component:

jsx
// components/DateCalendar.jsx
import React from "react";
import Calendar from "react-calendar";
import 'react-calendar/dist/Calendar.css';
import './calendarCustom.css'; // Optional: custom styles

export default function DateCalendar({ bookedDates = [], selectedDate, onDateChange }) {
  const isBooked = (date) =>
    bookedDates.some((d) => new Date(d).toDateString() === date.toDateString());

  return (
    <div className="max-w-sm mx-auto mb-10">
      <Calendar
        onChange={onDateChange}
        value={selectedDate}
        tileDisabled={({ date }) => isBooked(date)}
        tileClassName={({ date }) => isBooked(date) ? 'crossed-date' : null}
      />
    </div>
  );
}

3. Custom styles for crossed-out dates:

css
/* calendarCustom.css */
.crossed-date abbr {
  text-decoration: line-through;
  color: red;
}

4. Update your LandingPage.jsx:

Add the following inside LandingPage:

State Management

jsx
const [selectedDate, setSelectedDate] = useState(null); // shared date state
const [bookedDates, setBookedDates] = useState([

]); // replace with actual booked dates from backend

Update handleSubmit:

jsx
const handleSubmit = (e) => {
  e.preventDefault();
  const formData = new FormData(e.target);
  const location = formData.get("location");
  const checkin = formData.get("checkin");

  // Update shared date if checkin changed
  if (checkin) {
    setSelectedDate(new Date(checkin));
  }

  navigate(`/store?location=${location}&checkin=${checkin}`);
};

Sync calendar selection to booking form:

Add this to the date input:

jsx
<input
  type="date"
  name="checkin"
  value={selectedDate ? selectedDate.toISOString().split("T")[0] : ""}
  onChange={(e) => setSelectedDate(new Date(e.target.value))}
  className="border p-3 rounded-lg w-full md:w-1/4 focus:ring-2 focus:ring-red-400"
/>

5. Render the DateCalendar above map/listing:

Just before <PopularListing /> (or wherever your map will go):

jsx

  bookedDates={bookedDates}
  selectedDate={selectedDate}

/>

Final Behavior

  • The calendar disables/crosses out already booked dates.

  • Selecting a date updates the booking form.

  • Selecting a date from the form updates the calendar.

  • Both remain in sync.
